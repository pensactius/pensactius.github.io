1. Robótica con Arduino y MicroPython

(A partir de 11 años)

En este taller aprenderemos a programar diferentes dispositivos, pantallas OLED y sensores usando el lenguaje de programación Python (MicroPython) y Arduino.

  • Placa Arduino y ESP8266/32: Similitudes y diferencias
  • Programación en MicroPython y Arduino
  • Librerías en MicroPython y en Arduino
  • Uso de placa de prototipado
  • Estación Meteorológica: un proyecto que combina pantallas y sensores para poner en práctica lo aprendido. El proyecto se realizará en MicroPython y después se verá el equivalente con Arduino.

Horario de 9'10 a 11'10
Precio 75€

2. Técnicas de Fabricación Digital

(A partir de 11 años)

La próxima revolución industrial ya ha llegado y debemos estar preparados para ello. En este taller veremos como usar dos tecnologías increíbles para hacer realidad nuestras ideas: el corte láser y la impresión 3D. El objetivo de este taller será aprender a usar las nuevas tecnologías de fabricación digital y diseñar nuestros propios productos. Aplicaremos dichas técnicas para diseñar piezas para el sistema de construcción Stemfie, un sistema de construcción (parecido a Meccano) completamente gratuito en el que tú mismo te imprimes las piezas. Aprenderemos a diseñar piezas para Impresión 3D y también para corte láser. En el proceso veremos ventajas e inconvenientes de cada tecnología y los diferentes programas disponibles para cada una.

Horario de 11'30 a 13'30
Precio 75€ [INSCRIPCION]
